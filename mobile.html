 Login

All Successful logins will always make a new token and replace your existing one

/api/login (Normal Login)

Normal login using email and password (assuming account already exists)

  • email
  • password
  • device_type
On success On failure
/api/login (Social Login)

Social Login using facebook and google (if account already exists)

  • uid
  • provider
  • device_type
On Success:Same as Normal Login On failure
/api/login (Token Login)

This api is called if the app was not closed ,just reopened (still logged in) it will use the stored token on the device

  • token
  • device_type
On Success:Same as Normal Login On failure

 Assessment

/api/assessment/progresses

Return progress of all human assessment, gives null on a particular human assessment if record is not found

  • candidate_id
  • device_type (ANDROID / IOS)
  • token
On success On failure
/api/assessment/answers/questions

This api is called when answering an assessment, this will give the questions on that assessment or question type

  • question_type (1 - 8)
  • candidate_id
  • token
  • device_type
On success On failure
/api/assessment/answers/save

Call this api when answering and saving an assessment posting all question id and rate/answers

  • question_id +(one or more)
  • rate (1-5)
  • candidate_id
  • token
  • device_type
On success On failure
